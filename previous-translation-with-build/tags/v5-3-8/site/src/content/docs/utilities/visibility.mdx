---
title: Visibilidad (Visibility)
description: Controla la visibilidad de los elementos, sin modificar su display, con las utilidades de visibilidad.
---

Establece la `visibility` de los elementos con nuestras utilidades de visibilidad. Estas clases de utilidad no modifican el valor `display` en absoluto y no afectan al diseño: los elementos `.invisible` aún ocupan espacio en la página.

<Callout type="warning">
Los elementos con la clase `.invisible` se ocultarán *tanto* visualmente como para usuarios de tecnologías de asistencia/lectores de pantalla.
</Callout>

Aplica `.visible` o `.invisible` según sea necesario.

```html
<div class="visible">...</div>
<div class="invisible">...</div>
```

```scss
// Clase
.visible {
  visibility: visible !important;
}
.invisible {
  visibility: hidden !important;
}
```
