## Cómo funciona

Cambia el valor de la [propiedad `object-fit`](https://developer.mozilla.org/es/docs/Web/CSS/object-fit) con nuestras clases de utilidad `object-fit` responsivas. Esta propiedad le indica al contenido que llene el contenedor padre de varias formas, como conservando la relación de aspecto o estirándose para ocupar tanto espacio como sea posible.

Las clases para el valor de `object-fit` se nombran usando el formato `.object-fit-{value}`. Elige entre los siguientes valores:

- `contain`
- `cover`
- `fill`
- `scale` (para scale-down)
- `none`

## Ejemplos

Agrega la clase `object-fit-{value}` al [elemento reemplazado](https://developer.mozilla.org/es/docs/Web/CSS/Replaced_element):

## Responsivo

También existen variantes responsivas para cada valor de `object-fit` usando el formato `.object-fit-{breakpoint}-{value}`, para las siguientes abreviaturas de puntos de interrupción: `sm`, `md`, `lg`, `xl` y `xxl`. Las clases se pueden combinar para varios efectos según lo necesites.

<Example class="d-flex overflow-auto" code={`<Placeholder width="140" height="80" class="object-fit-sm-contain border rounded" text="Contain en sm" markup="img" />
<Placeholder width="140" height="80" class="object-fit-md-contain border rounded" text="Contain en md" markup="img" />
<Placeholder width="140" height="80" class="object-fit-lg-contain border rounded" text="Contain en lg" markup="img" />
<Placeholder width="140" height="80" class="object-fit-xl-contain border rounded" text="Contain en xl" markup="img" />
<Placeholder width="140" height="80" class="object-fit-xxl-contain border rounded" text="Contain en xxl" markup="img" />`} />

## Video

Las utilidades `.object-fit-{value}` y las utilidades responsivas `.object-fit-{breakpoint}-{value}` también funcionan en elementos `<video>`.

```html
<video src="..." class="object-fit-contain" autoplay></video>
<video src="..." class="object-fit-cover" autoplay></video>
<video src="..." class="object-fit-fill" autoplay></video>
<video src="..." class="object-fit-scale" autoplay></video>
<video src="..." class="object-fit-none" autoplay></video>
```
