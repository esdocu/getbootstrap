## Desbordamiento (Overflow)

Ajusta la propiedad `overflow` sobre la marcha con cuatro valores y clases predeterminados. Estas clases no son responsivas por defecto.

<div class="bd-example d-md-flex">
  <div class="overflow-auto p-3 mb-3 mb-md-0 me-md-3 bg-body-tertiary" style="max-width: 260px; max-height: 100px;">
    Este es un ejemplo del uso de <code>.overflow-auto</code> en un elemento con dimensiones de ancho y alto establecidas. Por diseño, este contenido se desplazará verticalmente.
  </div>
  <div class="overflow-hidden p-3 mb-3 mb-md-0 me-md-3 bg-body-tertiary" style="max-width: 260px; max-height: 100px;">
    Este es un ejemplo del uso de <code>.overflow-hidden</code> en un elemento con dimensiones de ancho y alto establecidas.
  </div>
  <div class="overflow-visible p-3 mb-3 mb-md-0 me-md-3 bg-body-tertiary" style="max-width: 260px; max-height: 100px;">
    Este es un ejemplo del uso de <code>.overflow-visible</code> en un elemento con dimensiones de ancho y alto establecidas.
  </div>
  <div class="overflow-scroll p-3 bg-body-tertiary" style="max-width: 260px; max-height: 100px;">
    Este es un ejemplo del uso de <code>.overflow-scroll</code> en un elemento con dimensiones de ancho y alto establecidas.
  </div>
</div>

```html
<div class="overflow-auto">...</div>
<div class="overflow-hidden">...</div>
<div class="overflow-visible">...</div>
<div class="overflow-scroll">...</div>
```

### `overflow-x`

Ajusta la propiedad `overflow-x` para afectar el desbordamiento de contenido horizontalmente.

    <div>Ejemplo de <code>.overflow-x-hidden</code></div>
    <div>en un elemento con dimensiones de ancho y alto establecidas.</div>
  </div>
  <div class="overflow-x-visible p-3 mb-3 mb-md-0 me-md-3 bg-body-tertiary w-100" style="max-width: 200px; max-height: 100px;white-space: nowrap;">
    <div>Ejemplo de <code>.overflow-x-visible</code></div>
    <div>en un elemento con dimensiones de ancho y alto establecidas.</div>
  </div>
  <div class="overflow-x-scroll p-3 bg-body-tertiary w-100" style="max-width: 200px; max-height: 100px;white-space: nowrap;">
    <div>Ejemplo de <code>.overflow-x-scroll</code> en un elemento</div>

```html

<div class="overflow-x-hidden">...</div>
<div class="overflow-x-visible">...</div>
<div class="overflow-x-scroll">...</div>
```

Usando variables Sass, puedes personalizar las utilidades de desbordamiento cambiando la variable `$overflows` en `_variables.scss`.
