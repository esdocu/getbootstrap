El componente auxiliar (helper) de enlace de icono modifica nuestros estilos de enlace predeterminados para mejorar su apariencia y alinear rápidamente cualquier combinación de icono y texto. La alineación se establece mediante estilos flexbox en línea y un valor `gap` predeterminado. Estilizamos el subrayado con un desplazamiento (offset) y un color personalizados. Los iconos se dimensionan automáticamente a `1em` para adaptarse mejor al tamaño de fuente (`font-size`) del texto asociado.

Los enlaces de iconos asumen que se utilizan los [Bootstrap Icons](https://icons.getbootstrap.com), pero puedes utilizar cualquier icono o imagen que desees.

<Callout>
Cuando los iconos son puramente decorativos, deben ocultarse a las tecnologías de asistencia utilizando `aria-hidden="true"`, como hemos hecho en nuestros ejemplos. Para los iconos que transmiten significado, proporciona una alternativa de texto adecuada agregando `role="img"` y un `aria-label="..."` adecuado a los SVG.
</Callout>

## Personalizar

Modifica el estilo de un enlace de icono con nuestras variables CSS de enlace, variables Sass, utilidades o estilos personalizados.

### Variables CSS

Modifica las variables CSS `--bs-link-*` y `--bs-icon-link-*` según sea necesario para cambiar la apariencia predeterminada.

Personaliza la propiedad `transform` al pasar el cursor sobrescribiendo la variable CSS `--bs-icon-link-transform`:

<Example code={`<a class="icon-link icon-link-hover" style="--bs-icon-link-transform: translate3d(0, -.125rem, 0);" href="#">
    <svg xmlns="http://www.w3.org/2000/svg" class="bi" viewBox="0 0 16 16" aria-hidden="true">
      <path d="M4 1.5H3a2 2 0 0 0-2 2V14a2 2 0 0 0 2 2h10a2 2 0 0 0 2-2V3.5a2 2 0 0 0-2-2h-1v1h1a1 1 0 0 1 1 1V14a1 1 0 0 1-1 1H3a1 1 0 0 1-1-1V3.5a1 1 0 0 1 1-1h1v-1z"/>
      <path d="M9.5 1a.5.5 0 0 1 .5.5v1a.5.5 0 0 1-.5.5h-3a.5.5 0 0 1-.5-.5v-1a.5.5 0 0 1 .5-.5h3zm-3-1A1.5 1.5 0 0 0 5 1.5v1A1.5 1.5 0 0 0 6.5 4h3A1.5 1.5 0 0 0 11 2.5v-1A1.5 1.5 0 0 0 9.5 0h-3z"/>
    </svg>
    Enlace de icono
  </a>`} />

    </svg>
  </a>`} />
