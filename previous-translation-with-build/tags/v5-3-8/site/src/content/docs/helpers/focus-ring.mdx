El ayudante (helper) `.focus-ring` elimina el contorno (`outline`) predeterminado en `:focus`, reemplazándolo por una sombra de caja (`box-shadow`) que se puede personalizar de forma más amplia. La nueva sombra se compone de una serie de variables CSS, heredadas desde el nivel `:root`, que se pueden modificar para cualquier elemento o componente.

## Ejemplo

Haz clic directamente en el siguiente enlace para ver el anillo de enfoque en acción, o haz clic dentro del siguiente ejemplo y luego presiona <kbd>Tab</kbd>.

<Example code={`<a href="#" class="d-inline-flex focus-ring py-1 px-2 text-decoration-none border rounded-2">
    Anillo de enfoque personalizado
  </a>`} />

## Personalizar

Modifica el estilo de un anillo de enfoque con nuestras variables CSS, variables Sass, utilidades o estilos personalizados.

### Variables CSS

Modifica las variables CSS `--bs-focus-ring-*` según sea necesario para cambiar la apariencia predeterminada.

<Example code={`<a href="#" class="d-inline-flex focus-ring py-1 px-2 text-decoration-none border rounded-2" style="--bs-focus-ring-color: rgba(var(--bs-success-rgb), .25)">
    Anillo de enfoque verde
  </a>`} />

`.focus-ring` establece estilos a través de variables CSS globales que se pueden sobrescribir en cualquier elemento padre, como se muestra arriba. Estas variables se generan a partir de sus contrapartes de variables Sass.

<ScssDocs name="root-focus-variables" file="scss/_root.scss" />

Por defecto, no existe `--bs-focus-ring-x`, `--bs-focus-ring-y` o `--bs-focus-ring-blur`, pero proporcionamos variables CSS con alternativas a los valores `0` iniciales. Modifícalos para cambiar la apariencia predeterminada.

<Example code={`<a href="#" class="d-inline-flex focus-ring py-1 px-2 text-decoration-none border rounded-2" style="--bs-focus-ring-x: 10px; --bs-focus-ring-y: 10px; --bs-focus-ring-blur: 4px">
    Anillo de enfoque con desplazamiento borroso
  </a>`} />

### API de utilidades Sass (Sass utilities API)

Además de `.focus-ring`, tenemos varias utilidades `.focus-ring-*` para modificar los valores predeterminados de la clase auxiliar. Modifica el color con cualquiera de nuestros [colores de tema]([[docsref:/customize/color#theme-colors]]). Ten en cuenta que las variantes clara y oscura pueden no ser visibles en todos los colores de fondo debido al soporte actual del modo de color.

<Example code={getData('theme-colors').map((themeColor) => `<p><a href="#" class="d-inline-flex focus-ring focus-ring-${themeColor.name} py-1 px-2 text-decoration-none border rounded-2">Enfoque ${themeColor.title}</a></p>`)} />

Las utilidades del anillo de enfoque se declaran en nuestra API de utilidades en `scss/_utilities.scss`. [Aprende a usar la API de utilidades.]([[docsref:/utilities/api#using-the-api]])
