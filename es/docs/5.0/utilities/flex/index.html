Habilitar comportamientos flex


Aplica utilidades de display para crear un contenedor flexbox y transforma los elementos hijos directos en elementos flex. Los contenedores y elementos flex se pueden modificar aún más con propiedades flex adicionales.




¡Soy un contenedor flexbox!


<div class="d-flex p-2 bd-highlight">I'm a flexbox container!</div>




¡Soy un contenedor flexbox en línea!


<div class="d-inline-flex p-2 bd-highlight">I'm an inline flexbox container!</div>

Dirección


Establece la dirección de los elementos flex en un contenedor flex con las utilidades de dirección. En la mayoría de los casos puedes omitir la clase horizontal aquí ya que el valor predeterminado del navegador es row. Sin embargo, puedes encontrarte con situaciones donde necesites establecer este valor explícitamente (como en layouts responsivos).


Usa .flex-row para establecer una dirección horizontal (el valor predeterminado del navegador), o .flex-row-reverse para iniciar la dirección horizontal desde el lado opuesto.

Crecer y reducir


Usa las utilidades .flex-grow-* para alternar la capacidad de un elemento flex de crecer para rellenar el espacio disponible. En el ejemplo de abajo, el elemento .flex-grow-1 usa todo el espacio disponible que puede, mientras permite a los dos elementos flex restantes su espacio necesario.





  
Elemento flex

  
Elemento flex

  
Tercer elemento flex




<div class="d-flex bd-highlight">
  <div class="p-2 flex-grow-1 bd-highlight">Flex item</div>
  <div class="p-2 bd-highlight">Flex item</div>
  <div class="p-2 bd-highlight">Third flex item</div>
</div>


Usa las utilidades .flex-shrink-* para alternar la capacidad de un elemento flex de reducirse si es necesario. En el ejemplo de abajo, el segundo elemento flex con .flex-shrink-1 se ve forzado a envolver su contenido a una nueva línea, "reduciéndose" para permitir más espacio para el elemento flex anterior con .w-100.





  
Elemento flex

  
Elemento flex

  <div class="p-2 w-100 bd-highlight">Flex item</div>
  <div class="p-2 flex-shrink-1 bd-highlight">Flex item</div>
</div>

Orden


Cambia el orden visual de elementos flex específicos con un puñado de utilidades de order. Solo proporcionamos opciones para hacer que un elemento sea primero o último, así como un reset para usar el orden del DOM. Como order toma cualquier valor entero de 0 a 5, añade CSS personalizado para cualquier valor adicional que necesites.

  <div class="order-3 p-2 bd-highlight">First flex item</div>
  <div class="order-2 p-2 bd-highlight">Second flex item</div>
  <div class="order-1 p-2 bd-highlight">Third flex item</div>
