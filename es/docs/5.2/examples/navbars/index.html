Ejemplos de barra de navegación

Este ejemplo es un ejercicio rápido para ilustrar cómo funcionan la barra de navegación y sus contenidos. Algunas barras de navegación se extienden a lo ancho del viewport, otras están confinadas dentro de un .container. Para el posicionamiento de las barras de navegación, consulta los ejemplos de superior y fija superior.

En el punto de interrupción más pequeño, el plugin collapse se usa para ocultar los enlaces y mostrar un botón de menú para alternar el contenido colapsado.
