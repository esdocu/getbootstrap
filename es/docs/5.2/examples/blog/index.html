Elementos HTML en línea

HTML define una larga lista de etiquetas en línea disponibles, cuya lista completa se puede encontrar en la Mozilla Developer Network.

  • Para poner texto en negrita, usa <strong>.
  • Para poner texto en cursiva, usa <em>.
  • Las abreviaturas, como HTML, deben usar <abbr>, con un atributo opcional title para la frase completa.
  • Las citas, como — Mark Otto, deben usar <cite>.
  • El texto eliminado debe usar <del> y el texto insertado debe usar <ins>.
  • El texto en superíndice usa <sup> y el texto en subíndice usa <sub>.

La mayoría de estos elementos son estilizados por los navegadores con pocas modificaciones por nuestra parte.
