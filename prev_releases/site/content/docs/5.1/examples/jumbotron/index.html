Ejemplo de Jumbotron

Jumbotron personalizado

Usando una serie de utilidades, puede crear este jumbotron, como el de versiones anteriores de Bootstrap. Echa un vistazo a los ejemplos a continuación para saber cómo puedes remezclarlo y cambiarle el estilo a tu gusto.

Cambiando el fondo

Cambia la utilidad de color de fondo y agregue una utilidad de color `.text-*` para mezclar el aspecto jumbotron. Luego, mezcla y combine con temas de componentes adicionales y más.

Añadir bordes

O manténlo ligero y agrega un borde para darle una definición adicional a los límites de su contenido. Asegúrate de mirar debajo del capó en el HTML de origen aquí, ya que hemos ajustado la alineación y el tamaño del contenido de ambas columnas para que tengan la misma altura.
