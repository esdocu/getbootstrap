Esta es la forma más básica de Bootstrap: archivos compilados para un uso rápido e integrado en casi cualquier proyecto web. Proporcionamos CSS y JS compilados (`bootstrap.*`), así como CSS y JS compilados y minificados (`bootstrap.min.*`). Los [mapas de fuentes (source maps)](https://web.dev/articles/source-maps) (`bootstrap.*.map`) están disponibles para su uso con las herramientas de desarrollo de ciertos navegadores. Los archivos JS agrupados (`bootstrap.bundle.js` y el minificado `bootstrap.bundle.min.js`) incluyen [Popper](https://popper.js.org/docs/v2/).

## Código fuente de Bootstrap

La descarga del código fuente de Bootstrap incluye los recursos CSS y JavaScript compilados, junto con el código fuente Sass, JavaScript y la documentación. Más específicamente, incluye lo siguiente y más:

```text
bootstrap/
├── dist/
│   ├── css/
│   └── js/
├── site/
│   └──content/
│      └── docs/
│          └── [[config:docs_version]]/
│              └── examples/
├── js/
└── scss/
```

Las carpetas `scss/` y `js/` son el código fuente de nuestro CSS y JavaScript. La carpeta `dist/` incluye todo lo enumerado en la sección de descarga compilada anterior. La carpeta `site/content/docs/` incluye el código fuente de nuestra documentación alojada, incluidos nuestros ejemplos en vivo del uso de Bootstrap.

Más allá de eso, cualquier otro archivo incluido proporciona soporte para paquetes, información de licencia y desarrollo.
