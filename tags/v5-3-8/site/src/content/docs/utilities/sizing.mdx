## Relativo al padre (Relative to the parent)

Las utilidades de ancho y alto se generan a partir de la API de utilidades en `_utilities.scss`. Incluye soporte para `25%`, `50%`, `75%`, `100%` y `auto` por defecto. Modifica esos valores según lo necesites para generar diferentes utilidades aquí.

<Example class="bd-example-flex" code={`<div class="w-25 p-3">Ancho 25%</div>
<div class="w-50 p-3">Ancho 50%</div>
<div class="w-75 p-3">Ancho 75%</div>
<div class="w-100 p-3">Ancho 100%</div>
<div class="w-auto p-3">Ancho auto</div>`} />

<Example class="bd-example-flex" code={`<div style="height: 100px;">
    <div class="h-25 d-inline-block" style="width: 120px;">Alto 25%</div>
    <div class="h-50 d-inline-block" style="width: 120px;">Alto 50%</div>
    <div class="h-75 d-inline-block" style="width: 120px;">Alto 75%</div>
    <div class="h-100 d-inline-block" style="width: 120px;">Alto 100%</div>
    <div class="h-auto d-inline-block" style="width: 120px;">Alto auto</div>
  </div>`} />

También puedes usar utilidades `max-width: 100%;` y `max-height: 100%;` según sea necesario.

<Example class="bd-example-flex" code={`<div style="width: 50%; height: 100px;">
    <div class="mw-100" style="width: 200%;">Ancho máximo 100%</div>
  </div>`} />

<Example class="bd-example-flex" code={`<div style="height: 100px;">
    <div class="mh-100" style="width: 100px; height: 200px;">Alto máximo 100%</div>
  </div>`} />

## Relativo a la ventana gráfica (Relative to the viewport)

También puedes usar utilidades para establecer el ancho y el alto en relación con la ventana gráfica (viewport).

```html
<div class="min-vw-100">Ancho mínimo 100vw</div>
<div class="min-vh-100">Alto mínimo 100vh</div>
<div class="vw-100">Ancho 100vw</div>
<div class="vh-100">Alto 100vh</div>
```
