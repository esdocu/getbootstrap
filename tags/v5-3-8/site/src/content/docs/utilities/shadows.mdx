## Ejemplos

Si bien las sombras en los componentes están deshabilitadas de forma predeterminada en Bootstrap y se pueden habilitar a través de `$enable-shadows`, también puedes agregar o quitar una sombra rápidamente con nuestras clases de utilidad `box-shadow`. Incluye soporte para `.shadow-none` y tres tamaños predeterminados (que tienen variables asociadas para que coincidan).

<Example class="overflow-hidden" code={`<div class="shadow-none p-3 mb-5 bg-body-tertiary rounded">Sin sombra</div>
<div class="shadow-sm p-3 mb-5 bg-body-tertiary rounded">Sombra pequeña</div>
<div class="shadow p-3 mb-5 bg-body-tertiary rounded">Sombra normal</div>
<div class="shadow-lg p-3 mb-5 bg-body-tertiary rounded">Sombra más grande</div>`} />
