## Imágenes responsivas

Las imágenes en Bootstrap se hacen responsivas con `.img-fluid`. Esto aplica `max-width: 100%;` y `height: auto;` a la imagen para que se escale con el ancho del padre.

## Picture

Si estás utilizando el elemento `<picture>` para especificar múltiples elementos `<source>` para una `<img>` específica, asegúrate de agregar las clases `.img-*` a la `<img>` y no a la etiqueta `<picture>`.

```html
<picture>
  <source srcset="..." type="image/svg+xml">
  <img src="..." class="img-fluid img-thumbnail" alt="...">
</picture>
```
