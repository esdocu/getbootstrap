## Ejemplo

Proporciona una opción para descartar o cerrar un componente con `.btn-close`. El estilo predeterminado es limitado, pero altamente ajustable. Modifica las variables Sass para reemplazar la `background-image` predeterminada. **Asegúrate de incluir texto para lectores de pantalla**, como hemos hecho con `aria-label`.

<Example code={`<button type="button" class="btn-close" aria-label="Close"></button>`} />

## Estado deshabilitado

Los botones de cierre deshabilitados cambian su `opacity`. También hemos aplicado `pointer-events: none` y `user-select: none` para evitar que se activen los estados activo y de desplazamiento (hover).

<Example code={`<button type="button" class="btn-close" disabled aria-label="Close"></button>`} />

## Variante oscura

<DeprecatedIn version="5.3.0" />

<Callout type="warning">
**¡Atención!** A partir de v5.3.0, la clase `.btn-close-white` está obsoleta. En su lugar, utiliza `data-bs-theme="dark"` para cambiar el modo de color del botón de cierre.
</Callout>

Agrega `data-bs-theme="dark"` al `.btn-close`, o a su elemento padre, para invertir el botón de cierre. Esto utiliza la propiedad `filter` para invertir la `background-image` sin anular su valor.

<Example class="bg-dark" code={`<div data-bs-theme="dark">
    <button type="button" class="btn-close" aria-label="Close"></button>
    <button type="button" class="btn-close" disabled aria-label="Close"></button>
  </div>`} />

### Variables

<AddedIn version="5.3.0" />

Los valores de las variables CSS se establecen a través de Sass, por lo que la personalización de Sass también es compatible.
