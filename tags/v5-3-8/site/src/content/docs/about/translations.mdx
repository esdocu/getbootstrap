---
title: Traducciones
description: Enlaces a sitios de documentación de Bootstrap traducidos por la comunidad.
---

import { getData } from '@libs/data'

Miembros de la comunidad han traducido la documentación de Bootstrap a varios idiomas. Ninguna está respaldada oficialmente y es posible que no siempre estén actualizadas.

<ul>
  {getData('translations').map((translation) => {
    return (
      <li><a href={translation.url} hreflang={translation.code} lang={translation.code} >{translation.description} ({translation.name})</a></li>
    )
  })}
</ul>

**No ayudamos a organizar ni a alojar traducciones, solo enlazamos a ellas.**

¿Terminaste una traducción nueva o mejor? Abre una solicitud de extracción para agregarla a nuestra lista.
