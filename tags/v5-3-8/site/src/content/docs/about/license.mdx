Bootstrap se publica bajo la licencia MIT y tiene derechos de autor de {new Date().getFullYear()}. Resumido en partes más pequeñas, se puede describir con las siguientes condiciones.

## Te exige:

- Mantener el aviso de licencia y derechos de autor incluido en los archivos CSS y JavaScript de Bootstrap cuando los uses en tus obras

## Te permite:

- Descargar y usar libremente Bootstrap, en su totalidad o en parte, para fines personales, privados, internos de la empresa o comerciales
- Usar Bootstrap en paquetes o distribuciones que crees
- Modificar el código fuente
- Otorgar una sublicencia para modificar y distribuir Bootstrap a terceros no incluidos en la licencia

## Te prohíbe:

- Responsabilizar a los autores y propietarios de la licencia por daños, ya que Bootstrap se proporciona sin garantía
- Responsabilizar a los creadores o titulares de los derechos de autor de Bootstrap
- Redistribuir cualquier parte de Bootstrap sin la debida atribución
- Usar cualquier marca propiedad de Bootstrap de cualquier manera que pueda declarar o implicar que Bootstrap respalda tu distribución
- Usar cualquier marca propiedad de Bootstrap de cualquier manera que pueda declarar o implicar que tú creaste el software de Bootstrap en cuestión

## No te exige:

- Incluir el código fuente del propio Bootstrap, o de cualquier modificación que le hayas realizado, en cualquier redistribución que realices y que lo incluya
- Enviar de vuelta al proyecto Bootstrap los cambios que realices a Bootstrap (aunque se agradecen tales comentarios)

La licencia completa de Bootstrap se encuentra [en el repositorio del proyecto]([[config:repo]]/blob/v[[config:current_version]]/LICENSE) para obtener más información.
