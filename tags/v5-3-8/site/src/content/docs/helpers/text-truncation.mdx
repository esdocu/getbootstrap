---
title: Truncado de texto (Text truncation)
description: Trunca cadenas largas de texto con puntos suspensivos.
toc: false
---

Para contenido más largo, puedes agregar la clase `.text-truncate` para truncar el texto con puntos suspensivos. **Requiere `display: inline-block` o `display: block`.**

<Example code={`<!-- Nivel de bloque -->
  <div class="row">
    <div class="col-2 text-truncate">
      Este texto es bastante largo y se truncará una vez que se muestre.
    </div>
  </div>

  <!-- Nivel en línea -->
  <span class="d-inline-block text-truncate" style="max-width: 150px;">
    Este texto es bastante largo y se truncará una vez que se muestre.
  </span>`} />
